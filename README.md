# shema_stroitelstva_vols

# 📡 Автоматизация ВОЛС - Расчет ТЭО и Схемы

Веб-приложение для автоматизации схем строительства волоконно-оптических линий связи (ВОЛС) и расчета технико-экономического обоснования (ТЭО).

## 🌟 Возможности

### Интерфейс и функционал
- **Верхняя панель**: Название проекта и поле для особых отметок (красным цветом)
- **Левая панель**: 
  - Выбор емкости кабеля (4, 8, 16, 48 волокон) с цветовой индикацией
  - Переключатель режима сортировки точек (по времени / по ближайшему соседу)
  - Drag-and-Drop зона для загрузки фотографий
  - Список всех точек на трассе
- **Центральная зона**: Интерактивная карта Leaflet.js с тайлами OpenStreetMap
- **Нижний блок**: Автоматическая генерация линейно-структурной схемы

### Обработка фотографий
- Извлечение GPS координат из EXIF метаданных
- Извлечение даты и времени съемки
- Автоматическое создание маркеров на карте
- Определение типа объекта по имени файла:
  - "УС" или "Клиент" → иконка здания
  - Остальные → стандартная точка

### Алгоритм трассировки
- Сортировка точек по времени съемки или по алгоритму ближайшего соседа
- Построение полилиний между точками с цветом в зависимости от емкости кабеля
- Расчет расстояний по формуле гаверсинусов
- Отображение расстояний на каждом пролете (Lфиз)
- Лимит 150м между опорами при геометрической сортировке

### Авто-генерация структурной схемы
- Прямоугольники для терминальных точек (КРН, КРС, первая/последняя)
- Круги для муфт (ОМ, Муфта)
- Скругленные прямоугольники для обычных опор
- Подписи марки кабеля и длины пролетов
- Отображение особых отметок красным шрифтом

### Экспорт
- Генерация единого PNG изображения высокого разрешения
- Включает: карту, структурную схему, статистику проекта
- Автоматическое скачивание файла с датой в имени

## 🚀 Развертывание на GitHub Pages

### Вариант 1: Быстрое развертывание
1. Создайте новый репозиторий на GitHub
2. Загрузите файл `index.html` в корень репозитория
3. Перейдите в Settings → Pages
4. В разделе "Source" выберите ветку `main` и папку `/ (root)`
5. Нажмите Save
6. Через 1-2 минуты приложение будет доступно по адресу: `https://<username>.github.io/<repo>/`

### Вариант 2: Клонирование готового репозитория
```bash
git clone https://github.com/yourusername/fols-automation.git
cd fols-automation
git push origin main
```

## 📋 Использование

### 1. Настройка проекта
- Введите название проекта в верхнем поле
- Укажите особые отметки (например: "ВАРИТЬ С ПРОСВЕТКОЙ")

### 2. Выбор параметров кабеля
- Выберите емкость кабеля в левой панели:
  - 🔵 4 волокна - голубой
  - 🟢 8 волокон - зеленый
  - 🟣 16 волокон - сиреневый
  - 🔴 48 волокон - красный

### 3. Загрузка фотографий
- Перетащите фотографии опор и муфт в зону загрузки
- Или кликните для выбора файлов
- Фотографии должны содержать GPS метаданные (EXIF)

### 4. Просмотр результатов
- Карта автоматически отобразит все точки с подписями
- Структурная схема сгенерируется внизу
- Статистика обновится в правом верхнем углу

### 5. Экспорт схемы
- Нажмите кнопку "Скачать исполнительную схему"
- Получите PNG файл со всей информацией о проекте

## 🛠️ Технологический стек

- **HTML5** - структура приложения
- **CSS3** - стили и адаптивный дизайн
- **Vanilla JavaScript** - логика приложения (без фреймворков)
- **Leaflet.js** - интерактивные карты
- **OpenStreetMap** - бесплатные картографические данные
- **exif-js** - чтение EXIF метаданных из фотографий
- **html2canvas** - генерация скриншотов и экспорт в PNG

## 📝 Требования к фотографиям

- Формат: JPEG, PNG
- Обязательно наличие GPS координат в EXIF метаданных
- Рекомендуется указывать дату съемки для правильной сортировки
- Имя файла становится подписью маркера на карте

## 🔒 Конфиденциальность

Все данные обрабатываются локально в браузере пользователя. Никакие фотографии или координаты не отправляются на сервер.
